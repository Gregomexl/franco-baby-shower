# OG Image Creation Instructions

## Required Specifications
- **Dimensions**: 1200 x 630 pixels
- **Format**: PNG or JPEG
- **File name**: `og-image.png`
- **Location**: `/public/og-image.png`

## Design Elements to Include

### Background
- Use the pistachio green gradient: `#E5EDD8`
- Add watercolor cloud elements for visual interest

### Main Content
1. **Top Section**:
   - "Baby Shower" text in serif font
   - Name "Franco" in script font (larger)

2. **Center**:
   - The teddy bear with balloons image (`baby_franco.png`)
   - Or use the family logo (`family.png`)

3. **Bottom Section**:
   - Date: "18 de Enero 2026"
   - Time: "11:00 A.M."
   - Location: "Edificio Aluna, Monterrey"

### Color Scheme
- **Primary green**: #7A9B84 (sage green)
- **Secondary**: #E5D4C1 (peach/beige)
- **Text**: #2D3436 (dark gray)
- **Background**: #E5EDD8 (pistachio green)

## Tools You Can Use

### Option 1: Canva (Easiest)
1. Create new design: 1200 x 630 px
2. Use baby shower templates
3. Customize with your colors and text
4. Download as PNG
5. Save to `/public/og-image.png`

### Option 2: Figma (Professional)
1. Create frame: 1200 x 630 px
2. Design with your brand elements
3. Export as PNG @1x
4. Save to `/public/og-image.png`

### Option 3: Online Tools
- [og-image-generator](https://www.opengraph.xyz/)
- [Social Image Generator](https://www.bannerbear.com/tools/social-media-image-generator/)

## Quick Test
After creating the image, test it:
1. Share your site URL on WhatsApp
2. Check if the image appears correctly
3. Verify on Facebook's Sharing Debugger: https://developers.facebook.com/tools/debug/
